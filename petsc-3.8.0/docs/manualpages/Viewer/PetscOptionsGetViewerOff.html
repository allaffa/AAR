PetscOptionsGetViewerOff

 does PetscOptionsGetViewer return a viewer?

Synopsis

#include "petscviewer.h" 
PetscErrorCode  PetscOptionsGetViewerOff(PetscBool *flg)
Logically Collective

Output Parameter

flg -whether viewers are returned.

Notes: Calling XXXViewFromOptions in an inner loop can be very expensive. This can appear, for example, when using many small subsolves.
