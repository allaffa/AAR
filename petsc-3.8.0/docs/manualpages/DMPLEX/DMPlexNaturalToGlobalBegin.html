DMPlexNaturalToGlobalBegin

 Rearranges a Vector in the natural order to the Global order.

Synopsis

#include "petscdmplex.h"   
PetscErrorCode DMPlexNaturalToGlobalBegin(DM dm, Vec nv, Vec gv)
Collective on dm

Input Parameters

dm - The distributed DMPlex
nv - The natural Vec

Output Parameters

gv -The global Vec

Note: The user must call DMPlexSetUseNaturalSF(dm, PETSC_TRUE) before DMPlexDistribute().

See Also

DMPlexDistribute(), DMPlexDistributeField(), DMPlexNaturalToGlobalBegin(),DMPlexGlobalToNaturalEnd()
