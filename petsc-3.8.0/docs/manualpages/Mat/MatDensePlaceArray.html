MatDensePlaceArray

 Allows one to replace the array in a dense array with an array provided by the user. This is useful to avoid copying an array into a matrix

Synopsis

#include "petscmat.h"  
PetscErrorCode  MatDensePlaceArray(Mat mat,const PetscScalar array[])
Not Collective

Input Parameters

mat - the matrix
array - the array in column major order

Notes

You can return to the original array with a call to MatDenseResetArray(). The user is responsible for freeing this array; it will not be freed when the matrix is destroyed.
