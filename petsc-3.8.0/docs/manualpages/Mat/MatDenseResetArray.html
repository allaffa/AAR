MatDenseResetArray

 Resets the matrix array to that it previously had before the call to MatDensePlaceArray()

Synopsis

#include "petscmat.h"  
PetscErrorCode  MatDenseResetArray(Mat mat)
Not Collective

Input Parameters

mat -the matrix

Notes

You can only call this after a call to MatDensePlaceArray()
