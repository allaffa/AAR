MatAYPX

 Computes Y = a*Y + X.

Synopsis

#include "petscmat.h"  
PetscErrorCode  MatAYPX(Mat Y,PetscScalar a,Mat X,MatStructure str)
Logically on Mat

Input Parameters

a - the PetscScalar multiplier
Y - the first matrix
X - the second matrix
str - either SAME_NONZERO_PATTERN, DIFFERENT_NONZERO_PATTERN or SUBSET_NONZERO_PATTERN

Keywords

matrix, add
