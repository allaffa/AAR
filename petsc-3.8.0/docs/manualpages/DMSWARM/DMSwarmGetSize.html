DMSwarmGetSize

 Retrives the total length of fields registered

Synopsis

#include "petscdmswarm.h"   
PETSC_EXTERN PetscErrorCode DMSwarmGetSize(DM dm,PetscInt *n)
Collective on DM

Input parameter

dm -a DMSwarm

Output parameter

n -the total length of each registered field

Note

This calls MPI_Allreduce upon each call (inefficient but safe)

See Also

DMSwarmGetLocalSize(), DMSwarmSetLocalSizes()
