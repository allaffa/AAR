PetscGLLElementGradientDestroy

 frees the gradient for a single 1d GLL element obtained with PetscGLLElementGradientCreate()

Synopsis

PetscErrorCode PetscGLLElementGradientDestroy(PetscGLL *gll,PetscReal ***AA,PetscReal ***AAT)
Not Collective

Input Parameter

gll - the nodes
AA - the stiffness element
AAT - the transpose of the element
