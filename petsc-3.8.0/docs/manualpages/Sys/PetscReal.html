PetscReal

 PETSc type that represents a real number version of PetscScalar

Notes: For MPI calls that require datatypes, use MPIU_REAL as the datatype for PetscScalar and MPIU_SUM, MPIU_MAX etc for operations. They will automatically work correctly regardless of the size of PetscReal

See PetscScalar for details on how to ./configure the size of PetscReal
