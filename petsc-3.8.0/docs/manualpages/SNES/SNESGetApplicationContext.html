SNESGetApplicationContext

 Gets the user-defined context for the nonlinear solvers.

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESGetApplicationContext(SNES snes,void *usrP)
Not Collective

Input Parameter

snes -SNES context

Output Parameter

usrP -user context

Fortran Notes: To use this from Fortran you must write a Fortran interface definition for this function that tells Fortran the Fortran derived data type that you are passing in as the ctx argument.

Keywords

SNES, nonlinear, get, application, context
