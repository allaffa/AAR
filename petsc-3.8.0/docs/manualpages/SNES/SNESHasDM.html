SNESHasDM

 Whether snes has dm

Synopsis

#include "petscsnes.h"  
PetscErrorCode  SNESHasDM(SNES snes,PetscBool *hasdm)
Not collective but all processes must return the same value

Input Parameter

snes -the nonlinear solver object

Output Parameter

hasdm -a flag indicates whether there is dm in snes
