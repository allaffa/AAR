DMDA_STENCIL_STAR

 "Star"-type stencil. In logical grid coordinates, only (i,j,k), (i+s,j,k), (i,j+s,k), (i,j,k+s) are in the stencil NOT, for example, (i+s,j+s,k)

Determines what ghost point values are brought over to each process; in this case the "corner" values are not brought over and hence should not be accessed locally
