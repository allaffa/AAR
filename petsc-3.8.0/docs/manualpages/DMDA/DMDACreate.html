DMDACreate

 Creates a DMDA object.

Synopsis

#include "petscdmda.h"   
PetscErrorCode  DMDACreate(MPI_Comm comm, DM *da)
Collective on MPI_Comm

Input Parameter

comm -The communicator for the DMDA object

Output Parameter

da -The DMDA object
