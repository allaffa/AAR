TSAdaptGetSafety

 Get safety factors

Synopsis

#include "petscts.h" 
PetscErrorCode TSAdaptGetSafety(TSAdapt adapt,PetscReal *safety,PetscReal *reject_safety)
Not Collective

Input Arguments

adapt - adaptive controller context

Ouput Arguments

safety - safety factor relative to target error/stability goal + reject_safety - extra safety factor to apply if the last step was rejected
