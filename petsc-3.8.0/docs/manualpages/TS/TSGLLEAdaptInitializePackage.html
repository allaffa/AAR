TSGLLEAdaptInitializePackage

 This function initializes everything in the TSGLLEAdapt package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to TSCreate_GLLE() when using static libraries.

Synopsis

#include "petscts.h" 
PetscErrorCode  TSGLLEAdaptInitializePackage(void)
