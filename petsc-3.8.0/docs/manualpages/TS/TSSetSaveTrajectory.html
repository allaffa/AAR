TSSetSaveTrajectory

 Causes the TS to save its solutions as it iterates forward in time in a TSTrajectory object

Synopsis

#include "petscts.h"  
PetscErrorCode  TSSetSaveTrajectory(TS ts)
Collective on TS

Input Parameters

ts -the TS context obtained from TSCreate()

Options Database

-ts_save_trajectory - saves the trajectory to a file
-ts_trajectory_type type- Note: This routine should be called after all TS options have been set

The TSTRAJECTORYVISUALIZATION files can be loaded into Python with $PETSC_DIR/bin/PetscBinaryIOTrajectory.py and MATLAB with $PETSC_DIR/share/petsc/matlab/PetscReadBinaryTrajectory.m

See Also

TSGetTrajectory(), TSAdjointSolve(), TSTrajectoryType, TSSetTrajectoryType()
