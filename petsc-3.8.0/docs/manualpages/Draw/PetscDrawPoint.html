PetscDrawPoint

 PetscDraws a point onto a drawable.

Synopsis

#include "petscdraw.h" 
PetscErrorCode  PetscDrawPoint(PetscDraw draw,PetscReal xl,PetscReal yl,int cl)
Not collective

Input Parameters

draw - the drawing context
xl,yl - the coordinates of the point
cl - the color of the point
