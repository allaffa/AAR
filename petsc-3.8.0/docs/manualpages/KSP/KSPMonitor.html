KSPMonitor

 runs the user provided monitor routines, if they exist

Synopsis

#include "petscksp.h" 
PetscErrorCode KSPMonitor(KSP ksp,PetscInt it,PetscReal rnorm)
Collective on KSP

Input Parameters

ksp - iterative context obtained from KSPCreate()
it - iteration number
rnorm - relative norm of the residual

Notes

This routine is called by the KSP implementations. It does not typically need to be called by the user.
